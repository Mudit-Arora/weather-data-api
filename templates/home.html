Weather Data API

URL Format to access data with station and date: http://127.0.0.1:5000/api/v1/station/date

Example: http://127.0.0.1:5000/api/v1/10/1988-10-25


URL Format to access data with station and year: http://127.0.0.1:5000/api/v1/yearly/station/year

Example: http://127.0.0.1:5000/api/v1/yearly/10/1988


URL Format to access data with station: http://127.0.0.1:5000/api/v1/date

Example: http://127.0.0.1:5000/api/v1/10
